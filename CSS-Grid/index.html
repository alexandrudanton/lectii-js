grid-gap | grid-template-columns

.example-1 { display: grid; grid-gap: 20px; grid-template-columns: 50px 50px 50px 50px; /* or grid-template-columns: repeat(4, 50px); */ }

Example 1

grid-template-columns ne ajuta sa setam numarul si latimea coloanelor din gridul nostru;

setand latimea pentru fiecare coloana, grid-ul stie automat si cate coloane vor fi afisate
.example-2 { display: grid; grid-gap: 20px 50px; grid-template-columns: 1fr 3fr 1fr 3fr; /* or grid-template-columns: repeat(2, 1fr 3fr); */ }

Example 2

unitatile de masura pe care le putem folosi pentru a seta latimea coloanelor din grid sunt aceleasi pe care le putem folosi cu width si in plus putem folosi fr

1 fr = (latimea gridului - suma latimilor coloanelor cu latime fixa) / numarul de unitati fr folosite in grid-template-columns
.example-3 { display: grid; grid-gap: 20px; grid-template-columns: 50px auto; }

Example 3

latimea auto pentru a doua coloana lateste coloana pe tot spatiul ramas liber resize the browser
.example-4 { display: grid; grid-gap: 20px; grid-template-columns: 50px auto 100px; }

grid-template-columns | grid-template-rows

.example-5 { display: grid; grid-gap: 20px; grid-template-columns: 50px 75px 100px; grid-template-rows: 50px 100px 150px; }

Example 5

folosind grid-template-rows putem sa specificam inaltimea fiecarei linii din grid; liniile pentru care nu se specifica inaltimea vor avea o inaltime auto conform cu continutul celulelor

grid-auto-rows

.example-6 { display: grid; grid-gap: 20px; grid-template-columns: repeat(2, 100px); grid-template-rows: 50px 75px; grid-auto-rows: 150px; }

Example 6

pentru a seta inaltimea pentru liniile care nu sunt "acoperite" de grid-template-rows se foloseste grid-auto-rows

grid-auto-flow

.example-7 { display: grid; grid-gap: 20px; grid-template-columns: 100px 50px; grid-auto-flow: column; }

Example 7

div-urile de la 3 la 10 ar fi trebuit sa fie afisate pe 2 coloane sub primele 2 div-uri, dar datorita grid-auto-flow: column; ele sunt afisate pe aceeasi linie cu primele doua

grid-column

.example-8 { display: grid; grid-gap: 20px; grid-template-columns: 100px 200px 50px; } .example-8 .item:nth-child(4n) { grid-column: span 3; }

Example 8

pentru a specifica numarul de coloane peste care sa se intinda o celula a gridului folosim grid-column: span [nr de coloane]

grid-column-start, grid-column-end | grid-row-start, grid-row-end

.example-9 { display: grid; grid-gap: 20px; grid-template-columns: repeat(10, 50px); } .example-9 .item:nth-child(2) { grid-column-start: 2; grid-column-end: 4; } .example-9 .item:nth-child(4) { grid-column-end: span 3; } .example-9 .item:nth-child(7) { grid-column-start: 10; grid-row-start: 1; grid-row-end: 3; } .example-9 .item:nth-child(18) { grid-column-end: span 3; grid-row-end: span 4; } .example-9 .item:nth-child(27) { grid-column-start: 6; grid-column-end: -1; } .example-9 .item:nth-child(32) { grid-column-start: 8; grid-column-end: -1; grid-row-start: 5; grid-row-end: span 3; } .example-9 .item:nth-child(44) { grid-column-start: 1; grid-column-end: -1; }

Example 9

o alternativa mult mai flexibila a grid-column: span [nr de coloane] este:

grid-column-start si grid-column-end pentru intinderea celulelor pe coloane
grid-row-start si grid-row-end pentru intinderea celulelor pe linii
